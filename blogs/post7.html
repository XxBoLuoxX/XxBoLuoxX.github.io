分享一下我知道的两个录播姬

随便往上瞎发电东西哈哈哈


录播姬

我自己最常用的一个录播姬
需要手动输入房间号添加
有点bug，比如可能录制文件提示编码错误，需要用【录播姬】→【工具箱】→【转封装】功能转为mp4格式就正常了
能录礼物，弹幕记录，美中不足的就是弹幕的用户名无法读取（因为没有登录功能，所以在b站那边估计就是游客模式） 上手挺简单，我现在目前还在用
详情可见以下连接

录播姬 录播姬 Github 页面


DDTV

比录播姬强，我目前还没试过
有扫码登录功能，可以办到一些录播姬办不到的东西，比如关注列表一键自动导入
